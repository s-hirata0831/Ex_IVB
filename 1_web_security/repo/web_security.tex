\section{目的}
現代ではゲームサーバ・通販サイトなどのサービスを，ネットワーク知識がなくても容易に解説できるまでになっている．
しかし，これらのサービスには脆弱性を意図せず含んでしまう場合があり，アカウント情報が盗まれるなどの被害が出る可能性がある．
そのため，本実験では実際に脆弱性を体験し，適切な対策を考察することを目的とする．
